DESCRIPTION


Web developed with JavaScript, using JQuery for the structure and JQuery UI for adding plugins.

The web have 4 pages, in which there are an universal clock, an accordion, an slider for images and a list of articles that can be added, in the main page.

Also there is a contact form with fields validation, and the possibility for log in and log out the page with a random user (no real users). The page have 3 color themes: red, blue and green, which can be changed on each moment, and have been configured using the local storage.
